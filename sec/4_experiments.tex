\section{Methodology}
\label{sec:methodology}

\subsection{Problem Formulation}

Let $D_{ID}$ and $D_{OoD}$ denote the in-distribution and out-of-distribution datasets, respectively. Given an input image $x$, an object detector $f_\theta$ produces predictions $f_\theta(x) = \{(b_i, c_i, s_i)\}_{i=1}^{N}$, where $b_i$ is the bounding box, $c_i$ is the class label, and $s_i \in [0,1]$ is the confidence score.

Under OoD inputs, detectors often assign high confidence to invalid regions:
\[
s_i^{OoD} \gg 0
\]

The objective is to reduce hallucinated detections while preserving in-distribution performance:
\begin{equation*}
\begin{aligned}
\min_{\theta}\;& \mathbb{E}_{x \sim D_{OoD}}[\text{FP}(f_\theta(x))] \\
\text{s.t.}\;& \mathbb{E}_{x \sim D_{ID}}[\text{Perf}(f_\theta(x))] \approx \text{constant}
\end{aligned}
\end{equation*}

\subsection{Overview}

Our approach consists of:

1. Training a baseline detector on $D_{ID}$
2. Fine-tuning with $D_{OoD}$ treated as background
3. Analyzing behavior using explainability metrics

\subsection{OoD Fine-tuning Strategy}

We extend standard training by incorporating OoD samples as background supervision:
\[
\mathcal{L} = \mathcal{L}_{YOLO}(D_{ID}) + \lambda \cdot \mathcal{L}_{bg}(D_{OoD})
\]

This enforces:
\[
s_i \rightarrow 0 \quad \text{for } x \sim D_{OoD}
\]

resulting in a more conservative objectness function and reduced hallucinations.

\subsection{Explainability and Behavioral Analysis}

We analyze how fine-tuning alters feature attribution and decision-making using multiple explainability methods.

\subsubsection{Grad-CAM}

Given feature maps $A^k$ from the last convolutional layer and a target score $y^c$, Grad-CAM weights are:
\[
\alpha_k^c = \frac{1}{Z} \sum_i \sum_j \frac{\partial y^c}{\partial A_{ij}^k}
\]

The Grad-CAM heatmap is computed as:
\[
L_{\text{Grad-CAM}}^c = \text{ReLU} \left( \sum_k \alpha_k^c A^k \right)
\]

This highlights regions contributing positively to the prediction.

\subsubsection{Occlusion Sensitivity}

Given an input image $x$, we define an occlusion operator $\mathcal{O}(x, r)$ that masks region $r$. The importance of region $r$ is:
\[
\Delta s_r = f_\theta(x) - f_\theta(\mathcal{O}(x, r))
\]

Higher $\Delta s_r$ indicates stronger dependence on region $r$.

\subsubsection{Insertion and Deletion Metrics}

Let $x$ be the input image and $M$ be the importance map.

\textbf{Insertion:} progressively add pixels in decreasing importance order:
\[
I(t) = f_\theta(x_t^{\text{insert}})
\]

\textbf{Deletion:} progressively remove important pixels:
\[
D(t) = f_\theta(x_t^{\text{delete}})
\]

The corresponding AUC scores are:
\[
\text{AUC}_{\text{ins}} = \int_0^1 I(t) \, dt, \quad
\text{AUC}_{\text{del}} = \int_0^1 D(t) \, dt
\]

Lower deletion AUC and higher insertion AUC indicate better explanations.

\subsubsection{IoU-based Alignment}

We measure overlap between Grad-CAM saliency regions $S$ and predicted bounding boxes $B$:
\[
\text{IoU} = \frac{|S \cap B|}{|S \cup B|}
\]

\textbf{Hallucination Rate (HR):}
\[
\text{HR} = \frac{\text{FP}}{\text{Total Detections}}
\]

\textbf{Reduction:}
\[
\text{Reduction}(\%) = \frac{\text{FP}_{\text{base}} - \text{FP}_{\text{method}}}{\text{FP}_{\text{base}}} \times 100
\]

\subsubsection{Frequency Bias Analysis (Proposed XAI Method)}

\paragraph{Frequency Estimation.}
For each detected region $x$ corresponding to a bounding box $b$, we compute a high-frequency energy measure using the Laplacian operator:
\begin{equation}
    e_f(x) = \frac{1}{|\Omega|} \sum_{p \in \Omega} |\Delta x(p)|,
\end{equation}
where $\Omega$ denotes the set of pixels in the region and $\Delta$ is the discrete Laplacian operator. This captures local high-frequency content such as edges and textures.

To ensure robustness across samples, we normalize frequency using a dataset-level statistic:
\begin{equation}
    \hat{f}(x) = \min\!\left(\frac{e_f(x)}{Q_{75}(e_f)},\; 2\right),
\end{equation}
where $Q_{75}$ is the 75th percentile of frequency energy values computed over all detection regions.

\paragraph{Objectness Proxy.}
Since explicit objectness scores are not directly available, we approximate objectness from confidence:
\begin{equation}
    o = \sqrt{s},
\end{equation}
which provides a monotonic transformation that stabilizes high-confidence responses.

\paragraph{Size Prior.}
Small regions are more prone to spurious detections. We define a size-dependent penalty based on bounding box area $a$:
\begin{equation}
\gamma(a) =
\begin{cases}
1.5 & \text{if } a < a_{\text{tiny}}, \\
1.0 & \text{otherwise}.
\end{cases}
\end{equation}

\paragraph{Post-hoc Calibration.}
We define a calibrated confidence score:
\begin{equation}
    s' = s \cdot \exp(-\alpha \hat{f}) \cdot \exp(-\beta o) \cdot \frac{1}{\gamma(a)},
\end{equation}
where $\alpha$ and $\beta$ control the strength of frequency and objectness suppression.

To ensure conservative behavior, we enforce:
\begin{equation}
    s' = \min(s', s),
\end{equation}
preventing artificial boosting of confidence.

\subsection{Evaluation Protocol}

We evaluate models on OoD inputs using:

\begin{itemize}
\item Detection statistics (counts, confidence)
\item IoU-based alignment
\item Pointing accuracy
\item Insertion/deletion AUC
\item Hallucination metrics
\end{itemize}

This enables a comprehensive analysis of both performance and internal behavior.

\section{Experiments and Results}
\label{sec:experiments_results}

\subsection{Effect of OoD Fine-tuning on Model Performance (Revised)}

\paragraph{Description}

In this experiment, we evaluate the impact of OoD fine-tuning on the robustness of object detectors under distribution shift. We compare a baseline YOLOv10s model trained on in-distribution (ID) data with a fine-tuned variant that is further trained using out-of-distribution (OoD) samples treated as background. The goal is to analyze whether such fine-tuning reduces hallucinated detections while maintaining or improving prediction confidence.

\paragraph{Experimental Setup}

We use the PASCAL VOC 2012 dataset as the in-distribution (ID) training set and the Caltech 256 dataset as the OoD source.

The baseline YOLOv10s model is trained for 50 epochs with an input resolution of 640×640 and batch size of 64. The fine-tuned model is initialized from the baseline and further trained for 20 epochs using a combined dataset where OoD samples are treated as background. Fine-tuning is performed with an input resolution of 512×512 and batch size of 32, using the default YOLO loss.

During evaluation, both models are tested on OoD images with a confidence threshold of 0.25. Detection statistics are computed over 100 OoD images.

\paragraph{Explanation and Analysis}

The results demonstrate several key effects of OoD fine-tuning.

\paragraph{1. Reduction in Hallucinated Detections}

The fine-tuned model reduces the average number of detections on OoD inputs (2.24 → 2.16), indicating improved suppression of false positives. This suggests that exposure to OoD samples during training helps the model better distinguish valid objects from irrelevant patterns.

\paragraph{2. Improved Confidence Calibration}

The average prediction confidence increases significantly (0.689 → 0.735) after fine-tuning. This indicates that the model becomes more confident in its predictions, likely due to improved feature discrimination and better calibration of objectness scores.

\paragraph{3. Decision Boundary Refinement}

Fine-tuning with OoD samples acts as a form of negative supervision, encouraging the model to assign lower objectness scores to regions that do not correspond to valid ID objects. This effectively reshapes the decision boundary, making it more conservative and reducing overgeneralization.

\paragraph{4. Stability of In-distribution Performance}

Despite the introduction of OoD samples, the model maintains stable training dynamics and competitive performance on ID data. This highlights that the fine-tuning process enhances robustness without sacrificing core detection capabilities.

\paragraph{5. Motivation for Explainability Analysis}

While detection count and confidence provide useful insights, they do not fully explain \textit{how} the model behavior changes. The observed improvements motivate a deeper investigation using explainability techniques, which we explore in the next experiment.

\subsection{Experiment 2}
\paragraph{Confidence Calibration Effect}

\textbf{Description.}
This analysis examines how post-hoc calibration alters the distribution of detection confidence scores under OoD conditions. The goal is to evaluate whether calibration reduces overconfident predictions that contribute to hallucinated detections.

\textbf{Experimental Setup.}
Confidence scores are collected from all detections produced on OoD images before and after applying the proposed calibration. A confidence threshold of 0.25 is used, and histograms are constructed to compare the distribution of raw and calibrated scores.

\textbf{Results.}
The calibrated distribution exhibits a clear shift toward lower confidence values. In the raw detector, a large number of detections lie in the high-confidence range, whereas after calibration, the density of such predictions is significantly reduced.

\textbf{Explanation.}
This shift indicates that the detector is inherently overconfident when operating on OoD inputs. The proposed calibration mechanism effectively suppresses these overconfident predictions, leading to a more conservative and reliable confidence distribution. This behavior directly contributes to the observed reduction in false positives.

\paragraph{Frequency--Objectness Bias}

\textbf{Description.}
This analysis investigates the relationship between frequency characteristics of detected regions and their corresponding objectness scores. The goal is to understand whether high-frequency patterns systematically influence detector confidence.

\textbf{Experimental Setup.}
For each detected bounding box in OoD images, the frequency energy is computed over the corresponding image patch. Objectness is approximated from the detection confidence. A scatter plot is constructed to visualize the relationship between frequency energy and objectness.

The scatter plot reveals a noticeable trend where regions with higher frequency energy tend to produce higher objectness scores. This indicates that texture-rich regions are more likely to trigger strong detector responses.

\textbf{Explanation.}
This behavior suggests that the detector is biased toward high-frequency patterns, treating them as indicative of object presence even when no meaningful object exists. In OoD settings, such textures often arise from background clutter, leading to hallucinated detections. By penalizing high-frequency regions during calibration, this bias is reduced, resulting in improved robustness.

\subsection{Occlusion Sensitivity Analysis for Explainability}

\subsubsection{Description}
We analyze the effect of out-of-distribution (OoD) data on object detection models using \textbf{occlusion sensitivity analysis}. This method helps identify which regions of an image contribute most to model predictions by systematically masking parts of the input. Our goal is to understand why the baseline model produces hallucinated detections and how fine-tuning with OoD data as background improves model behavior.

\subsubsection{Experimental Setup}
Occlusion is applied using a sliding window approach with a fixed patch size (e.g., $32 \times 32$) and stride (e.g., 16). For each image, local regions are masked to generate perturbed inputs, and the change in detection confidence is recorded to construct importance heatmaps.

We evaluate explanations using:
\begin{itemize}
    \item \textbf{Heatmaps}: visualize the spatial importance of different regions, highlighting areas (red) that contribute most to the model's prediction.
    \item \textbf{Insertion AUC}: measures how quickly confidence increases when important regions are added.
    \item \textbf{Deletion AUC}: measures how quickly confidence drops when important regions are removed.
    \item \textbf{Pointing Game Accuracy}: checks whether the most important point lies within the ground truth bounding box.
\end{itemize}

All metrics are averaged over 50 test images. Heatmaps are also visually compared to assess spatial alignment.

\subsubsection{Results}
Quantitatively, the fine-tuned model outperforms the baseline across all metrics. It achieves higher insertion AUC, lower deletion AUC, and improved pointing game accuracy, indicating more reliable and faithful explanations. Qualitatively, heatmaps from the baseline model highlight irrelevant texture regions, whereas the fine-tuned model focuses on semantically meaningful object areas.
